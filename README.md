# 🚀 Playwright Automation Framework

## Overview

This project is built using **Playwright with JavaScript** for end-to-end web UI automation testing.

## Technologies Used

- Playwright
- JavaScript
- Node.js
- Git
- GitHub

## Features

- UI Automation Testing
- Assertions
- Built-in Locators
- Multiple Element Handling
- OpenCart Automation
- Homepage Testing

## Project Structure

```
homepage-tests/
tests/
package.json
playwright.config.js
```

## Installation

```bash
npm install
```

## Run Tests

```bash
npx playwright test
```
